# LaporBukti - Pelaporan Fasilitas Kampus/Umum

Aplikasi **LaporBukti** adalah aplikasi mobile berbasis Android yang dirancang untuk memudahkan warga kampus/umum melaporkan kerusakan fasilitas secara cepat, akurat, dan transparan. Aplikasi ini dibangun dengan pendekatan **Offline-First**, sehingga tetap dapat digunakan meskipun tanpa koneksi internet.

Proyek ini dibuat untuk memenuhi **Tugas Akhir (UAS) Mata Kuliah Pemrograman Bergerak**.

1.  **Laporan Kerusakan:** Input judul, kategori, dan deskripsi kerusakan dengan mudah.
2.  **Bukti Foto:** Ambil foto langsung dari kamera/galeri sebagai bukti valid.
3.  **Lokasi GPS Akurat:** Deteksi titik lokasi kejadian secara otomatis (High Accuracy).
4.  **Mode Offline:** Simpan laporan di database lokal (Room) saat tidak ada sinyal.
5.  **Dashboard Admin:** Fitur khusus admin untuk memantau, memfilter, dan mengubah status laporan.
6.  **Statistik Visual:** Grafik Pie Chart untuk memantau persentase laporan masuk.
